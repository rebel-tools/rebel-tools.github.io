Introducing Jibber

Jibber was born out of frustration with working with JSON data. It was extremely infurating to debug errors using Xcode's small console window. We envisioned a product that would seamlessly integrate into all your projects and allow you to work with JSON easily.
Jibber works by using Bonjour to establish a "Zeroconf" . Therefore we are able to establish the port number that we can use to connect to your phone over the local wifi and thus the Mac client is able to connect and receive the network requests that you are making
We use "method swizzling" in order to 'hijack' the network requests that you make through NSURLSessionTasks. This we ensure that we are able to cover almost every body as this would work even with the popular "AFNetworking" class as it uses NSURLSessionTask underneath.
